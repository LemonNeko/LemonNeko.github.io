搭建 github 博客

创建仓库

新建一个名为 <你的用户名>.github.io 的仓库，比如你的用户名为 admin，则你新建的仓库名应为 admin.github.io，请注意必须使用你自己的用户名，将来访问你的网站就直接是 https://admin.github.io 了

绑定域名

  • 不绑定域名也是可以的，可以直接使用 xxx.github.io 来访问
  • 注册域名可以去阿里云或者腾讯云
  • 推荐使用 CNAME 方式解析，主机记录选 www, 线路默认，记录值填 <你的用户名>.github.io
