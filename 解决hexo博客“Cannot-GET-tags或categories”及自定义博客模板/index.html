自定义博客模板

  • 在站目录下可以看见 scaffolds 文件夹

  • 打开可以看见如下文件结构

  • 文章可以有如下属性

    • layout 文章的布局，可以取值 post（默认值）或 page，可以通过修改 _config.yml 中的 default_layout 参数来指定默认布局。
    • title 文章的标题
    • date 创建日期，文件的创建日期
    • updated 修改日期，文件的修改日期
    • comments 是否开启评论，默认值 true
    • tags 标签
    • categories 分类
    • permalink url 中的名字，默认值文件名
